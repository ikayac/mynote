# Apps 運用ルール

このディレクトリでは、趣味のアプリ開発やプログラミング学習のプロジェクトを管理します。

## フォルダ構成
プロジェクトは常に日付をプレフィックスとしたフォルダ名で作成します。

- 形式: `YYYY-MM-DD_[プロジェクト名]`
- 例: `2026-01-01_rust-learning`

古いスクリプトや、特定のプロジェクトに属さない単発の試作は `apps/archive/` 内に同様の形式で配置します。

## 推奨スタック
- **Rust**: 高パフォーマンスが必要なツールや低レイヤ開発
- **Julia**: 数値計算・データ解析（`study/experiments` との使い分けに注意）
- **Python**: 汎用スクリプト、プロトタイピング

## Git 運用
- 新しいプロジェクトを開始する際は、必ずブランチを作成してください。
- 作業完了後、マージする前にビルドやテストが通ることを確認してください。
